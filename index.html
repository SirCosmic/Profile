Importante

JS

JavaScript (JS) é uma linguagem de programação leve, interpretada e orientada a objetos que é amplamente utilizada para adicionar interatividade a páginas web, mas também pode ser usada em outros ambientes como Node.js, Apache CouchDB e Adobe Acrobat.

CSS

CSS (Cascading Style Sheets), ou "Folha de Estilo em Cascata" em português, é uma linguagem que define a aparência de um documento HTML. Ele controla aspectos como cores, fontes, tamanhos, layout e outros elementos visuais de uma página web. Em resumo, o CSS é responsável por dar estilo e visual a um site, enquanto o HTML define a estrutura e o conteúdo.

HTML

HTML (HyperText Markup Language) é a linguagem de marcação que forma a base de conteúdo da web, responsável por estruturar páginas web e definir como o conteúdo deve ser apresentado no navegador. É como o esqueleto de uma página web, que define o significado e a estrutura do texto, imagens, vídeos e outros elementos.
